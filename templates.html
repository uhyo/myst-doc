Renderers

my-static is equipped with built-in renderers for jade (.jade), ejs (.ejs) and dustjs (.dust).

It also has renderers for Sass (.sass, .scss) that compiled Sass code into CSS, and for some static files (.css, .js, .html) that just copy files into output directory.
